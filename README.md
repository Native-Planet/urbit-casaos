# Urbit on Umbrel

This is a repository for running an Urbit Pier on Umbrel

## Install on Umbrel

To install on umbrel for testing 
1. Make a directory for the yaml script:  `mkdir -p ~/umbrel/apps/urbit-umbrel`
2. cd into the umbrel directory: `cd ~/umbrel`
3. Run the install app script: `scripts/app install urbit-umbrel`
4. Navigate to http://umbrel.local:8090/launch to view the control screen. Once your pier is booted landscape is acessible at http://umbrel.local:8090
